
# Лабораторна робота 10

Цей репозиторій містить рішення завдань з налаштування HTTP-серверів на Node.js.

## 10.1: Створення базового сервера
Реалізація простого HTTP-сервера.
- **Мета:** Запуск сервера з використанням `process.env.PORT` та обробка базових запитів.

## 10.3: Production Hardening
Захищена версія сервера, оптимізована для використання в продуктовому середовищі.

### Функціонал:
- **Безпекові заголовки:** Додано `X-Content-Type-Options: nosniff`, `X-Frame-Options: DENY` та `Referrer-Policy: no-referrer`.
- **Підтримка CORS:** Налаштовано `Access-Control-Allow-Origin: *` та обробку `OPTIONS` запитів для `/health`.
- **Обробка помилок:** Маршрут `/boom` повертає статус `500` з повідомленням "Internal Server Error", не зупиняючи роботу сервера.
- **Перевірка стану:** Маршрут `/health` повертає `{"ok": true}`.
